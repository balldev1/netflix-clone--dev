# NEXTAUTH_JWT_SECRET เป็นคีย์สำหรับเข้ารหัสและถอดรหัส JSON Web Token (JWT) 
# ที่ใช้ในการตรวจสอบและรักษาสถานะการเข้าสู่ระบบของผู้ใช้ระหว่างเซสชัน 
# คุณสามารถกำหนดค่านี้เป็นสตริงสุ่มหรือคีย์ที่คุณเลือกได้เพื่อรักษาความปลอดภัยของ JWT.

# NEXTAUTH_SECRET เป็นค่าลับที่ใช้ในการเข้ารหัสและถอดรหัสข้อมูลอื่นๆ เ
# ช่น รหัสการเข้ารหัส cookie และข้อมูลอื่นๆที่จำเป็นใน NextAuth. 
# ค่านี้ควรเป็นสตริงสุ่มหรือคีย์ที่เป็นความลับและเป็นค่าที่ไม่สามารถเดาได้.

overflow-x-hidden และ overflow-y-auto คือการกำหนดการเลื่อนแนวนอนและแนวตั้งของเนื้อหาภายในองค์ประกอบ 
โดย overflow-x-hidden จะซ่อนการเลื่อนแนวนอนเมื่อเนื้อหายาวเกินกว่าความกว้างขององค์ประกอบ 
และ overflow-y-auto จะเปิดใช้งานการเลื่อนแนวตั้งเมื่อเนื้อหายาวเกินกว่าความสูงขององค์ประกอบ 

เมื่อใช้ scale-100 องค์ประกอบจะมีขนาดเดิมที่ไม่มีการเปลี่ยนแปลงใดๆ
 ในขณะที่ scale-0 จะทำให้องค์ประกอบหายไปเนื่องจากมีขนาดศูนย์
